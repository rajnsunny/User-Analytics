# User-Analytics

## SETUP
-  clone the Repo
-  npm install
-  install Mobx library
-  install FakerJS
-  install react-chartjs-2
-  Now run the build by: `npm run dev`

## TECHNOLOGIES
 1. `TypeScript | React | Vite`
 2. chart : `react-chartjs-2`
 3. data : `FakerJS`
 4. Data Storage: `MobX Store`
